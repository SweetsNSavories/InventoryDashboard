Data Dictionary & Relationships

The schema is designed for multi-tenant scalability, mapping logical cloud environments to physical solution objects and their constituent assets.

Environment Parent ➔ Contains N Solutions and N Assets (1:N)
Solution Context ➔ Groups multiple Assets via Solution ID (1:N)
Metadata Store ➔ `gov_metadata` stores the raw API JSON for long-term technical audit
